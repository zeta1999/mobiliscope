Le dossier "data processing/v3/FrenchCities" contient les scripts nécessaires à la préparation des données
des villes FRANCAISES à partir des enquêtes ménages déplacements. 

Les étapes de traitements des données peuvent être reproduites pour Montpellier et sa région en utilisant le jeu de données brutes 
mis à disposition par Montpellier Méditerrannée Métropole sous license ODbL et les scripts R de l'équipe du Mobiliscope.

Contenu des sous-dossiers :
- "opendata" : données sources issues de l'EDGT Montpellier 2014

- "processing" : 
	- "scriptsr": scripts R utilisés pour la préparation des données du Mobiliscope pour les villes FRANCAISES (exemple de Montpellier)
	- "data" : bases de données utilisées ou produites sous R (exemple de Montpellier)
	- "shp" : couches d'information géographique utilisées ou produites sous R (exemple de Montpellier)
	- "txt" : fichiers csv utilisés ou produits sous R
